`ProseLintBear <https://github.com/coala-analyzer/coala-bears/tree/master/bears/natural_language/ProseLintBear.py>`_
=================

Lints the file using ``proselint``.

`Supported Languages <../README.rst>`_
-----

* Natural Language



Dependencies
------------

.. code-block:: bash

    $ pip install proselint==0.3.*



Can Detect
----------

* Formatting
* Grammar
* Spelling
* Syntax
