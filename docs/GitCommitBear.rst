**GitCommitBear**
=================

Checks the given commit body.

`Supported Languages <../README.rst>`_
-----

* Git

Settings
--------

+---------------------------------+-----------------------------------+
| Setting                         |  Meaning                          |
+=================================+===================================+
|                                 |                                   |
| ``allow_empty_commit_message``  | Whether empty commit messages are |
|                                 | allowed or not.                   |
|                                 |                                   |
+---------------------------------+-----------------------------------+

\* denotes required param

Can Detect
----------

* Formatting
