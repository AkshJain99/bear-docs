**CPPCheckBear**
================

Report possible security weaknesses for C/C++.
For more information, consult <https://github.com/danmar/cppcheck>.

`Supported Languages <../README.rst>`_
-----

* C
* C++



Can Detect
----------

* Redundancy
* Security
* Smell
