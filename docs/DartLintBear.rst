**DartLintBear**
================

Checks the code with ``dart-linter``.

This bear expects dart commands to be on your ``PATH``. Please ensure
/path/to/dart-sdk/bin is in your ``PATH``.

`Supported Languages <../README.rst>`_ :
-----

* Dart
