**VultureBear**
===============

Checks Python code for unused variables and functions using ``vulture``.

See <https://bitbucket.org/jendrikseipp/vulture> for more information.

`Supported Languages <../README.rst>`_
-----

* Python
* Python 3



Can Detect
----------

* Unused Code
