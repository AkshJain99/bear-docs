**XMLBear**
===========

Checks the code with ``xmllint``.

`Supported Languages <../README.rst>`:
-----

* XML

--------

+-----------------+----------------------------------------------+
| Setting         |  Meaning                                     |
+=================+==============================================+
|                 |                                              |
| ``xml_schema``  | ``W3C XML Schema`` file used for validation. +
|                 |                                              |
+-----------------+----------------------------------------------+
|                 |                                              |
| ``xml_dtd``     | ``Document type Definition (DTD)`` file or   |
|                 | url used for validatio                       |
|                 |                                              |
+-----------------+----------------------------------------------+

\* denotes required param
