**JSComplexityBear**
====================

Calculates cyclomatic complexity using ``cr``.

`Supported Languages <../README.rst>`_
-----

* JavaScript

Settings
--------

+-------------------+------------------------------------------------------+
| Setting           |  Meaning                                             |
+===================+======================================================+
|                   |                                                      |
| ``cc_threshold``  | Threshold value for cyclomatic complexity (Optional, |
|                   | defaults to '10'.)                                   |
|                   |                                                      |
+-------------------+------------------------------------------------------+


Can Detect
----------

* Complexity
