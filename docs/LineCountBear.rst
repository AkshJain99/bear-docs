**LineCountBear**
=================

Count the number of lines in a file and ensure that they are smaller than a given size.

`Supported Languages <../README.rst>_`:
-----

* All

--------

+---------------------------+---------------------------------+
| Setting                   |  Meaning                        |
+===========================+=================================+
|                           |                                 |
| ``max_lines_per_file``\*  | Number of lines allowed per fil +
|                           |                                 |
+---------------------------+---------------------------------+

\* denotes required param
