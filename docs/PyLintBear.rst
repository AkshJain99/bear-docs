**PyLintBear**
==============

Checks the code with pylint. This will run pylint over each file
separately.

`Supported Languages <../README.rst>`_
-----

* Python
* Python 2
* Python 3



Can Detect
----------

* Duplication
* Formatting
* Security
* Syntax
* Unused Code
