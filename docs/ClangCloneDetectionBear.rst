**ClangCloneDetectionBear**
===========================

Checks the given code for similar functions that are probably redundant.

`Supported Languages <../README.rst>`_
-----

* C
* C++
* CUDA
* Objective-C
* Objective-C++
* OpenCL
* OpenMP

|                           |                                       |

|                           | hav                                   |
|                           |                                       |
+---------------------------+---------------------------------------+
